# SupplyChain Guard CLI

SupplyChain Guard CLI is a lightweight cybersecurity tool that helps developers identify supply chain security risks in software dependencies before they are installed or deployed.

The tool analyzes dependency files such as `requirements.txt` and `package.json` to detect known vulnerabilities, suspicious packages, typosquatting attempts, and other potential security threats. It provides a simple risk assessment report that enables developers to make informed decisions about the packages they use.

## Key Objectives

- Improve software supply chain security
- Detect risky and vulnerable dependencies
- Identify possible typosquatting attacks
- Provide dependency risk scoring
- Help developers secure projects early in the development lifecycle

SupplyChain Guard CLI aims to act as a first line of defense against malicious or compromised open-source packages.
